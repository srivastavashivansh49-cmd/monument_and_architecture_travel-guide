## 📌 Project Overview

**Monument and Architecture Travel Guide** is a Django web application designed to help users explore monuments, architectural places, and travel destinations.  
The system includes user registration, login, admin management, place listings, booking features, contact management, and profile handling.

This project is suitable for academic submissions, portfolio presentation, and beginner-to-intermediate Django learning.

---

## ✨ Features

### 👤 User Features

- User registration and login
- View available monuments and architecture destinations
- View detailed information about places
- Book places/destinations
- View booking history
- Edit user profile
- Change password
- Submit contact/enquiry form

### 🛠️ Admin Features

- Admin login
- Admin dashboard
- Add new places/rooms
- Edit existing place details
- Delete places
- View all bookings
- Change booking status
- View registered users
- Delete users
- View contact messages
- Delete contact messages
- Change admin password
